# Telecom Churn Analysis 📊

This beginner-level project uses Python and pandas to explore **customer churn behavior** in the telecom industry.

### 📁 Dataset
The dataset contains customer details such as contract type, monthly charges, tenure, and whether they have churned or not.

### 📌 Key Analysis:
- Overall churn distribution (Yes/No)
- Churn rate by contract type
- Tenure vs churn behavior
- Monthly charges vs churn likelihood

### 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

### 📈 Outcome
- Found that customers with month-to-month contracts have the highest churn.
- Shorter tenure and higher monthly charges are correlated with churn.
